# Assets

Project assets including images, icons, and media files.

## Files

- **`spotify-resolver.png`** - Main project icon/logo (256x256px)
  - Used in README.md header
  - Spotify-themed design
  - Can be customized (see docs/ICON-README.md)

## Usage in Documentation

The icon is displayed in README.md:

```markdown
<div align="center">
  <img src="assets/spotify-resolver.png" alt="Spotify Album Resolver" width="256" height="256">
</div>
```

## Adding New Assets

When adding new assets:
1. Use descriptive filenames
2. Optimize file sizes (use compression)
3. Document usage in this README
4. Update main README if asset is user-facing

## Asset Guidelines

- **Images**: PNG or JPEG, optimized
- **Icons**: 256x256px minimum, transparent backgrounds preferred
- **Screenshots**: Retina-ready (@2x)
- **Max file size**: Keep under 500KB when possible
